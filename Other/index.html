What is Authoring Systems?

  • A program which has pre-programmed elements for the development of interactive multimedia software .
  • Authoring systems vary widely in orientation, capabilities, and learning curve.
  • Whether you realize it or not, authoring is actually just a speeded-up form of programming; you don't need to know the intricacies of a programming language, or worse, an application programming interface (API), but you do need to understand how programs work (Merrill, 1985).

    What is Authoring Systems?

    An application development environment for non programmers. The only difference between such tools with a programming language is that a nonprogrammer can learn to use such a tool in a short space of time (relative to that taken to learn a conventional programming language, such as C++ or Pascal) and use it to produce high-quality applications.
